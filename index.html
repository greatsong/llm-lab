임베딩 핵심
임베딩 공간에서는 의미가 비슷한 단어끼리 가까이 모입니다. 단어를 클릭하면 다른 단어와의 유사도를 확인할 수 있습니다. 임베딩 행렬도 가중치이며, 역전파로 학습됩니다.
문장 선택
어텐션 히트맵 — 단어를 클릭하면 집중 대상이 강조됩니다
어텐션 해석
어텐션 = 각 단어가 문맥을 이해하기 위해 다른 단어에 부여하는 중요도. 색이 진할수록 더 많이 집중합니다. 어텐션 가중치도 학습으로 최적화됩니다.

LLM 작동 원리
LLM = 어텐션으로 문맥을 파악한 뒤, 다음에 올 단어의 확률을 예측하는 기계. Temperature가 낮으면 가장 높은 확률의 단어만 선택하고, 높으면 다양한 단어가 선택될 수 있습니다. 인공지능의 역사는 문제해결의 역사이고, 아이디어의 역사이고, 알고리즘의 역사입니다.

문제 → 아이디어 → 알고리즘

알고리즘: 토큰화 → 임베딩 → 어텐션 → 트랜스포머
